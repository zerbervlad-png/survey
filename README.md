# Платформа проведения опросов

Внутренний инструмент для создания и проведения опросов среди сотрудников. Администратор может создавать опросы с разными типами вопросов, загружать списки сотрудников, отслеживать прогресс и выгружать детальную статистику.

## Возможности
- Создание опросов с вопросами разных типов (текст, одиночный выбор, множественный выбор)
- Загрузка списка сотрудников из Excel
- Автоматическое скрытие завершённых опросов от пользователей
- Двухшаговый процесс прохождения (выбор ФИО → анимация → вопросы)
- Детальная статистика в админке с визуализацией распределения ответов
- Экспорт результатов в Excel (с детализацией и статистикой по вариантам)
- Полностью адаптивный дизайн в стиле Apple

## Технологии
- Python + Flask
- SQLite
- HTML/CSS (чистый, без фреймворков)
